# Brief de Creative de Ad

## Principio Fundamental

EL CREATIVE DE AD EXCELENTE DETIENE EL SCROLL EN BAJO 2 SEGUNDOS — CADA ELEMENTO (IMAGEN, HEADLINE, COPY, CTA) DEBE TRABAJAR JUNTO PARA ENTREGAR UN MENSAJE CLARO QUE COMPELA ACCIÓN.

---

## Cuándo Usar

- Crear brief de creative detallado para producción de ad
- Escribir múltiples variaciones de copy para diferentes plataformas y formatos
- Definir dirección visual, dimensiones y guías de brand para activos de ad
- Briefear designer, videographer o herramienta de AI image sobre requisitos de creative de ad

---

## Fase 1: Brief

### Inputs Requeridos

| Input | Qué Preguntar | Default |
|---|---|---|
| **Producto/oferta** | "¿Qué estás publicitando?" | Sin default — debe ser proporcionado |
| **Plataformas** | "¿Dónde correrán estos ads?" | Facebook + Instagram |
| **Objetivo** | "¿Qué debe lograr el ad?" | Conversiones |
| **Audiencia destino** | "¿Para quién es este ad? Sé específico." | Sin default — debe ser proporcionado |
| **Guías de brand** | "¿Tienes colores, fonts o reglas de estilo?" | Sin guías formales |
| **Número de variaciones** | "¿Cuántas versiones de creative necesitas?" | 3-5 variaciones |
| **Formatos** | "¿Imágenes estáticas, video, carousel, o todos?" | Imágenes estáticas + video |

---

## Fase 2: Estrategia de Creative

### Ángulos de Mensajería

Define 3-5 ángulos de mensajería para probar:

```
## Ángulos de Creative

**Ángulo 1: Problema-Solución**
Message: "[Pain point] → [Tu producto lo arregla]"
Emotion: Relief, frustración → resolución

**Ángulo 2: Prueba Social**
Message: "[X clientes / resultado específico] logrado con [producto]"
Emotion: Confianza, FOMO

**Ángulo 3: Beneficio Primero**
Message: "[Outcome específico que quieren] — aquí cómo"
Emotion: Deseo, aspiración
```

---

## Anti-Patrones

- No crear el brief sin audiencia destino confirmada — "todos" no es audiencia
- No incluir más de un CTA por variación de creative
- No mezclar múltiples ángulos de mensajería en un mismo ad
- No producir creatives sin especificaciones de dimensión por plataforma
- No omitir el brief visual — el copy solo no basta para briefear producción
- No asumir que el designer conoce el tono de marca sin documentarlo

---

## Outputs Esperados

- Brief de creative completo con dirección visual por variación
- 3-5 ángulos de mensajería definidos y listos para probar
- Especificaciones de formato y dimensiones por plataforma
- Copy de ad escrito para cada variación (headline + texto primario + CTA)
- Instrucciones de producción para designer, videographer o herramienta AI image
