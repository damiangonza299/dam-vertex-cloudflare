# Oferta Tripwire

## Principio Fundamental

EL TRABAJO DEL TRIPWIRE NO ES GANANCIA — ES CAMBIAR A ALGUIEN DE LEAD A COMPRADOR, PORQUE LOS COMPRADORES SON 10X MÁS PROPENSOS A COMPRAR NUEVAMENTE QUE QUIENES NUNCA HAN COMPRADO.

---

## Cuándo Usar Este Skill

- Diseñar oferta bajo-ticket ($7-$47) que convierta leads en compradores primerizos
- Crear copy de página de ventas y plan de entrega para producto tripwire
- Mapear el tripwire en escalera de productos más grande con rutas de upsell
- Definir psicología de precio para que la oferta se sienta como un robo

**NO USES** este skill para precios de oferta principal, lead magnets gratis, o páginas de ventas alto-ticket. Esto es específicamente para ofertas bajo-costo diseñadas para crear compradores.

---

## Fase 1: Brief

### Inputs Requeridos

| Input | Qué Preguntar | Default |
|---|---|---|
| **Oferta principal** | "¿Cuál es tu producto/servicio principal (el que el tripwire lleva)?" | Sin default — debe ser proporcionado |
| **Lead magnet existente** | "¿Tienes ya un lead magnet o freebie?" | No — crearemos uno |
| **Audiencia** | "¿Quién es tu cliente ideal?" | Sin default — debe ser proporcionado |
| **Nicho/industria** | "¿En qué nicho o mercado operas?" | Sin default — debe ser proporcionado |
| **Precio de oferta principal** | "¿Cuánto cuesta tu producto principal?" | Sin default — debe ser proporcionado |

**PUNTO DE CONTROL: No procedas sin conocer la oferta principal — el tripwire debe ser un puente hacia ella.**

## Fase 2: Diseño de Tripwire

### Concepto de Producto

El tripwire debe:
1. **Resolver un problema específico rápido** — no solución compreh ensiva
2. **Estar directamente relacionado a oferta principal** — compradores deben querer el producto más grande después
3. **Alto valor percibido, precio bajo** — debe sentirse como un robo
4. **Entrega instantánea** — sin espera

### Framework de Tripwire

```
Nombre del producto: [Nombre que implica valor y velocidad]
Formato: [Paquete de plantillas / Mini-curso / Archivo swipe / Herramienta]
Precio: $[7-47]
Valor percibido: $[100-500]
Entrega: Descarga instantánea / acceso inmediato

La Victoria Rápida: ¿Qué resultado específico obtiene el comprador?
La Brecha: ¿Qué necesitarán a continuación? (lleva a oferta principal)
El Puente: ¿Cómo este producto lleva naturalmente a querer más?
```

## Fase 3: Posicionamiento en Escalera de Producto

### Mapa de Escalera

```
## Escalera de Producto

Nivel 0 — Lead Magnet (Gratis):
[Nombre] — [Qué resuelve en 5 min]
↓ (lleva a)

Nivel 1 — Tripwire ($7-47):
[Nombre] — [Victoria rápida específica]
↓ (lleva a)

Nivel 2 — Core Offer ($97-497):
[Nombre] — [Transformación completa]
↓ (lleva a)

Nivel 3 — High-Ticket / VIP ($997+):
[Nombre] — [Resultado premium con apoyo]
```

### Regla del Puente

Cada nivel debe crear el deseo del nivel siguiente:
- El lead magnet muestra el problema
- El tripwire da la primera solución / herramienta
- La core offer da el sistema completo
- El high-ticket da implementación / soporte

## Fase 4: Psicología de Precio del Tripwire

### Selección de Precio

| Precio | Cuándo Usarlo | Psicología |
|---|---|---|
| **$7** | Lead magnet mejorado, muy bajo riesgo | "No puedo no probarlo" |
| **$17** | Plantilla única, guía rápida | Precio impar, parece específico |
| **$27** | Mini-curso, swipe file | Sweet spot entre valor y riesgo |
| **$37-47** | Kit completo, herramienta funcional | Límite superior del tripwire |

**Regla del Robo:** El comprador debe sentir que está obteniendo $100-500 de valor por $7-47. Si no lo siente, el precio está mal o el producto tiene poco valor percibido.

### Anclaje de Precio

```
Precio de anclaje visible: ~~$[97-197]~~
Tu precio hoy: $[7-47]

Justificación del descuento (opcional):
- "Precio de lanzamiento"
- "Para nuevos miembros"
- "Incluido con tu descarga gratuita"
```

## Fase 5: Copy de Página de Ventas del Tripwire

### Estructura de Página (Short-Form Sales Page)

```
## Página de Ventas Tripwire — [Nombre del Producto]

**Headline:**
"[Resultado específico] en [tiempo corto] — sin [obstrucción principal]"

**Subheadline:**
"[Descripción de una línea de qué es + para quién]"

**El Problema (2-3 oraciones):**
"[Problema que el comprador reconoce inmediatamente]"

**La Solución (2-3 oraciones):**
"[Producto] te da [qué específicamente]. Ya no necesitas [problema]."

**Qué Incluye:**
✓ [Componente 1] — [Por qué importa]
✓ [Componente 2] — [Por qué importa]
✓ [Componente 3] — [Por qué importa]
(Valor total: $[X])

**La Victoria Rápida:**
"En [tiempo], tendrás [resultado específico]."

**Precio:**
~~$[precio de anclaje]~~ → Solo $[precio real]

**CTA:**
"Sí, quiero [nombre del producto] por $[precio]"
(Acceso inmediato — descarga instantánea)

**Garantía (opcional pero recomendado):**
"Si no [resultado mínimo] en [tiempo], te devuelvo cada centavo."

**P.S.:**
"Recuerda: [urgencia o recordatorio del valor vs precio]"
```

## Fase 6: Upsell Post-Compra

### Secuencia de Upsell Inmediata

```
## Página de Upsell — Inmediatamente Después de Compra del Tripwire

**Headline:**
"Espera — antes de descargar [Tripwire], una sola cosa más:"

**Transición:**
"Ya tomaste el primer paso con [Tripwire]. Si quieres [resultado más grande], tengo algo especial para ti ahora mismo."

**La Core Offer presentada como siguiente lógico:**
"[Core Offer] es el sistema completo que [resultado de la core offer]."

**Por qué ahora:**
"Como nuevo cliente de [Tripwire], tienes acceso a precio especial — solo en esta página."

**Precio especial:**
~~$[precio regular]~~ → $[precio especial] (ahoras $[X])

**CTA:**
"Sí, agregar [Core Offer] a mi orden"
(o "No gracias, solo quiero [Tripwire] por ahora")
```

---

## Anti-Patrones

- **NO** poner el tripwire en el mismo nivel de complejidad que la core offer — debe ser simple y rápido
- **NO** precio mayor a $47 — pierde el efecto "no hay riesgo"
- **NO** hacer el tripwire tan bueno que el cliente no necesite la core offer
- **NO** entrega con fricción — si el comprador tarda más de 60s en acceder, el efecto se pierde
- **NO** upsell inmediato sin dar primero acceso al tripwire — frustración mata conversión
- **NO** describir el tripwire como "introducción básica" — debe sentirse como valor completo
