# Generador de Copia de Ad

## Principio Fundamental

TODO AD DEBE SER NATIVO A SU PLATAFORMA, GOLPEAR LÍMITES DE CARACTERES EXACTAMENTE, E IMPULSAR UNA ACCIÓN CLARA ÚNICA.

---

## Cuándo Usar Este Skill

- Usuario necesita ad copy pagado para Facebook, Instagram, Google o LinkedIn
- Usuario está lanzando una campaña nueva y necesita variaciones de creative
- Usuario quiere variaciones de prueba A/B para ads existentes
- Usuario necesita copy para un creative de ad
- Usuario pide recomendaciones de targeting de audiencia para ads pagados

---

## Fase 1: Brief

Reúne estos inputs antes de escribir una sola palabra de copy.

1. **Producto/Servicio** — ¿Qué estás publicitando?
2. **Audiencia Destino** — ¿Quién es el cliente ideal?
3. **Plataforma(s)** — Facebook/Instagram, Google Search, Google Display, LinkedIn
4. **Oferta/CTA** — ¿Qué acción? (prueba gratis, compra, reserva, descarga)
5. **Objetivo** — ¿Conciencia (top of funnel) o Conversión (bottom of funnel)?
6. **Landing Page URL** — ¿Dónde envían los ads?
7. **Brand Voice** — Profesional, casual, audaz, empático, autoridad
8. **Canva Graphics** — ¿Usuario quiere imágenes de ad generadas?

**PUNTO DE CONTROL: No procedas a Fase 2 sin producto, audiencia, al menos una plataforma, y CTA confirmados.**

---

## Fase 2: Escribe Ads Primarios

Escribe un ad primario por plataforma solicitada usando especificaciones de formato. Cuenta cada carácter.

### Fórmulas de Ad Copy

| Fórmula | Estructura | Mejor Para |
|---|---|---|
| PAS | Dolor > Agitar > Resolver | Conversión, audiencias conscientes de problema |
| AIDA | Atención > Interés > Deseo > Acción | Conciencia, audiencias frías |
| Antes/Después | Forma antigua vs Nueva forma | Lanzamientos de producto, diferenciadores |
| Prueba Social | Resultado + prueba + CTA | Ofertas alto-confianza, testimonios |
| Urgencia | Presión de tiempo/escasez + CTA | Ofertas limitadas, lanzamientos |

---

## Fase 3: Especificaciones por Plataforma

### Facebook / Instagram Ads

```
## Ad Facebook/Instagram — [Variación A/B/C]

**Fórmula usada:** [PAS / AIDA / Prueba Social / Urgencia]
**Objetivo:** [Conversiones / Tráfico / Leads]
**Temperatura de audiencia:** [Fría / Tibia / Caliente]

**Texto Primario:** (máx 125 chars para preview completo, hasta 500 funcional)
"[Línea de apertura — dolor o beneficio impactante]

[Desarrollo — agitar problema o ampliar beneficio]

[Solución — producto como respuesta]

[Prueba social o garantía breve si aplica]

[CTA claro y único]"

**Headline:** (máx 40 chars)
"[Beneficio específico o resultado concreto]"

**Descripción:** (máx 30 chars — visible en algunos placements)
"[Refuerzo del headline o urgencia]"

**CTA Button:** [Comprar ahora / Más información / Registrarse / Enviar mensaje]
```

### Google Search Ads

```
## Ad Google Search — [Variación A/B/C]

**Keyword Intent:** [Informacional / Navegacional / Transaccional]

**Headline 1:** (máx 30 chars)
"[Keyword principal + beneficio]"

**Headline 2:** (máx 30 chars)
"[Diferenciador o CTA]"

**Headline 3:** (máx 30 chars)
"[Prueba social, precio o urgencia]"

**Description 1:** (máx 90 chars)
"[Beneficio principal + feature de respaldo]"

**Description 2:** (máx 90 chars)
"[Manejo de objeción + CTA]"

**URL Final:** [URL de landing page]
**Display Path:** domain.com/[keyword-relevante]/[subtema]
```

### LinkedIn Ads

```
## Ad LinkedIn — [Variación A/B/C]

**Formato:** [Single Image / Carousel / Message Ad]
**Objetivo:** [Lead Gen / Website Visit / Brand Awareness]

**Intro Text:** (máx 150 chars para preview, hasta 600 chars)
"[Hook profesional — problema de negocio o resultado de negocio]

[Contexto y relevancia para audiencia B2B]

[Solución + diferenciador]

[CTA específico]"

**Headline:** (máx 70 chars)
"[Beneficio de negocio cuantificado o promesa concreta]"

**CTA Button:** [Más información / Registrarse / Descargar / Aplicar ahora]
```

---

## Fase 4: Variaciones A/B

### Framework de 3 Variaciones por Plataforma

**Variación A — Ángulo Problema/Dolor:**
Abre con el problema más agudo. Agita. Presenta el producto como alivio.

**Variación B — Ángulo Beneficio/Resultado:**
Abre con el outcome deseado. Describe la vida mejor. Producto como camino.

**Variación C — Ángulo Prueba Social/Credibilidad:**
Abre con datos, testimonios o número de clientes. Producto validado por otros.

---

## Fase 5: Recomendaciones de Targeting

### Sugerencias de Audiencia por Tipo de Producto

```
## Targeting Recomendado

**Audiencia Principal (Intereses):**
- [Interés 1 directamente relacionado]
- [Interés 2 — comportamiento de compra]
- [Interés 3 — demografía complementaria]

**Audiencia Lookalike:**
- Fuente: [Lista de compradores / visitantes de checkout / top 25% LTV]
- Porcentaje: 1-2% (escalar a 3-5% si funciona)

**Exclusiones:**
- Compradores de últimos 30/60 días (en campañas de adquisición)
- Empleados / competidores si aplica
- Edades fuera de rango del cliente ideal

**Notas de Targeting:**
- [Observación específica sobre la audiencia de este producto]
```

---

## Anti-Patrones

- **NO** escribir copy genérico que funcione para "cualquier producto" — el copy específico convierte más
- **NO** usar más de un CTA por ad — "Compra O Aprende Más O Regístrate" = 0 acción
- **NO** ignorar los límites de caracteres de la plataforma — se trunca y pierde impacto
- **NO** escribir en primera persona de empresa ("Nosotros ofrecemos...") — hablar al cliente
- **NO** sepultar el beneficio principal — debe estar en el primer 20% del texto
- **NO** usar jerga técnica si la audiencia es amplia

---

## Outputs Esperados

Al completar este skill, el usuario tiene:
1. Copy completo para 3 variaciones A/B por plataforma
2. Especificaciones de formato respetadas por plataforma
3. Sugerencias de targeting de audiencia
4. Notas de brief para designer si aplica
5. Criterios para evaluar qué variación gana
